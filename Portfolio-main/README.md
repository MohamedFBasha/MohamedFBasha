# Ibrahim Hendawi - Portfolio Website

A modern, responsive portfolio website showcasing Ibrahim Hendawi's skills, projects, and experience in Data Science, Machine Learning, and Artificial Intelligence.

## 🌟 Features

- **Responsive Design**: Optimized for all devices (desktop, tablet, mobile)
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Interactive Elements**: Hover effects, scroll animations, and smooth transitions
- **Performance Optimized**: Fast loading times and optimized assets
- **Accessibility**: WCAG compliant with proper semantic HTML
- **SEO Friendly**: Structured data and meta tags for better search visibility

## 🛠️ Technologies Used

- **HTML5**: Semantic markup structure
- **CSS3**: Modern styling with flexbox/grid, animations, and responsive design
- **JavaScript (ES6+)**: Interactive functionality and smooth user experience
- **Font Awesome**: Icon library for visual elements
- **Google Fonts**: Inter font family for typography

## 🚀 Getting Started

1. **Clone or Download**: Get the project files
2. **Open**: Simply open `index.html` in your web browser
3. **Customize**: Edit the content to match your information

### Local Development

For a better development experience, you can use a local server:

```bash
# Using Python 3
python -m http.server 8000

# Using Node.js (if you have http-server installed)
npx http-server

# Using PHP
php -S localhost:8000
```

Then visit `http://localhost:8000` in your browser.

## 🔧 Customization Guide

### Updating Personal Information
1. Edit the hero section in `index.html`
2. Update contact details throughout the file
3. Modify social media links

### Adding Projects
1. Copy a project card in the projects section
2. Update the content with your project details
3. Add appropriate technology tags

### Styling Changes
1. Modify CSS custom properties in `:root` for colors
2. Update font families in the CSS file
3. Adjust spacing and sizing variables

### Adding New Sections
1. Create the HTML structure
2. Add corresponding styles in CSS
3. Update navigation menu
4. Add JavaScript if needed for interactivity
